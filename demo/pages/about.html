csa is a minimalistic client framework

what it's capable of:

  • single page application - navigation
  • basic reactivity
  • one, global data tunnel is used for reactivity. this means while it's harder to name things without conflicts, it's easier to share reactive data on the whole html document

what's more to come:

  • spa - scroll, page restoration
  • advanced reactivity
